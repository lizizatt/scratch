# Ground truth / test fixtures

- **calibration.gcode** – Axis-sweep GCODE (X, then Y, then Z) with known segment lengths and feedrates. Generated by `scripts/generate_instrument_gcode.py`.
- **manifest.json** – Per-segment list: axis, length_mm, feedrate_mm_per_min, duration_sec. Use to align timeline with recordings or tests.

Regenerate:

```bash
python scripts/generate_instrument_gcode.py -o data/ground_truth/calibration.gcode --manifest data/ground_truth/manifest.json
```

Optional files (if you use them): `recording.wav`, `reference.mid`, `gcode_params.json`, `timing_params.json` – not required for the melody-optimize pipeline.
